# Hisham Khashman - Portfolio

Single-page portfolio built with Next.js + Tailwind.

## Quick Start

```bash
npm install
npm run dev
```

## Update Content

- Replace `public/me.svg` with your photo (keep the filename or update `components/Hero.tsx`).
- Replace project images in `public/projects/` and update paths in `data/projects.ts`.
- Update LinkedIn URL in `data/profile.ts` (currently `https://www.linkedin.com/in/your-handle`).
- Update CV by replacing `public/cv.pdf`.

## Structure

- `app/page.tsx`: single-page layout
- `components/`: UI sections
- `data/`: profile + projects data
- `public/`: images + CV
